pydistort
=========

The package ``pydistort`` is composed of the following functions, classes, and modules:

- ``pydistort.Distortion`` base class provides functions for managing image distortion.
- ``pydistort.ZernikeDistortion`` class provides functions for managing Zernike polynomials.
- ``pydistort.utils`` module provides utility functions for managing image distortion.
