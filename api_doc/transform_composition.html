pydistort.core.TranformComposition#

class pydistort.core.TransformComposition(transformations: Sequence[Transform])[source]#

A class to represent the composition of multiple transformations.

It inherits from the pydistort.core.Transform class and overrides the _transform and _inverse_transform methods to apply the composition of transformations.

If the transformations to represent is \(T_n (T_{n-1}(...(T_1(X))))\), the transformations must be given in the order of application, i.e. from the first to the last transformation.

transformations = [Transform1(), Transform2(), ..., TransformN()]
Parameters:

transformations (Sequence[Transform]) – A list of transformations to be composed. Each transformation must be an instance of the Transform class or its subclasses.

property Nparams: int#

The total number of parameters in the composition of transformations. This is the sum of the number of parameters of each transformation in the composition.

_inverse_transform(points: ndarray, *, dx: bool = False, dp: bool = False, **kwargs) → Tuple[ndarray, ndarray | None, ndarray | None][source]#

Apply the inverse transformation to the given points.

This method applies the inverse of each transformation in the composition sequentially to the input points.

The jacobian matrices with respect to the input points and the parameters are computed using the chain rule.

Parameters:

  • points (numpy.ndarray) – The input points to be transformed. Shape (Npoints, output_dim).

  • dx (bool, optional) – If True, compute the Jacobian of the transformed points with respect to the input points. Default is False.

  • dp (bool, optional) – If True, compute the Jacobian of the transformed points with respect to the parameters of the transformation. Default is False.

  • **kwargs – Additional keyword arguments for the transformations.

Returns:

A tuple containing:

  • transformed_points: The transformed points of shape (Npoints, input_dim).

  • jacobian_dx: The Jacobian matrix with respect to the input points of shape (Npoints, input_dim, output_dim) if dx is True, otherwise None.

  • jacobian_dp: The Jacobian matrix with respect to the parameters of the transformation of shape (Npoints, input_dim, Nparams) if dp is True, otherwise None.

Return type:

Tuple[numpy.ndarray, Optional[numpy.ndarray], Optional[numpy.ndarray]]
